1. IoT Landscape

Mobile

    LTE Long-Term Evolution

    LTE is a standard for high-speed wireless communication for mobile devices and data terminals, based on the GSM/EDGE and UMTS/HSPA technologies. It increases the capacity and speed using a different radio interface together with core network improvements. The standard is developed by the 3GPP (3rd Generation Partnership Project).

    protocol, wireless, mobile
    Beacon

    Beacons are small devices that enable relatively accurate location within a narrow range (like indoors). Beacons periodically transmit small amounts of data within a range of approximately 70 meters, and are often used for indoor location technology.

    location, mobile, navigation
    GPS Global Positioning System

    GPS provides geolocation and time information to a receiver anywhere on or near the Earth. It does not require the user to transmit any data, and it operates independently of any telephonic or internet reception.

    navigation, transport, wearables, mobile, location
    Gyroscope

    A gyroscope is a type of sensor useful for measuring orientation. Electronic, microchip-packaged MEMS gyroscopes are commonly found in consumer electronics devices.

    mobile, wearables
    Pedometer

    A pedometer is a device, usually portable and electronic or electromechanical, that counts each step a person takes by detecting the motion of the person's hands or hips.

    mobile, wearables, health
    RFID Radio-frequency Identification

    Radio-frequency identification (RFID) uses electromagnetic fields to automatically identify and track tags, containing electronically stored information, attached to objects. There are passive and active tags.

    mobile, radio, tracking, industry
